利用宝塔面板安装Sprov-ui v2ray

· 2019-06-23 ·

系统使用Debian即可 在你的Linux服务器上依次运行以下命令：

apt-get install git

git clone https://github.com/fafayaa/sprov-ui

cd sprov-ui

bash install.sh

bash /root/sprov-ui/sprov-ui.sh && chmod +x /usr/bin/sprov-ui

然后安装宝塔面板

wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && bash install.sh

其他系统请射入
