记一次安装v2ray带Sprov-ui面板

2019-06-23

首先我们需要选购一个vps 这里推荐老毛子justhost的11软妹币/月小型vps
地址 (访问可能较慢）
配置如下

虚拟化：KVM

  • 核心：1
  • RAM：512 Mb
  • 硬盘：5 Gb
  • 流量：无限

这家俄罗斯主机商现已经支持 支付宝

推荐服务器地址选择阿德曼（新西伯利亚）

系统使用Debian即可 我这里安装的是纯净的Debian9

在你的Linux服务器上依次运行以下命令：

apt-get install git

git clone https://github.com/fafayaa/sprov-ui

cd sprov-ui

bash install.sh

bash /root/sprov-ui/sprov-ui.sh && chmod +x /usr/bin/sprov-ui

至此访问服务器ip+设置的端口就可以进入Sprov面板啦
